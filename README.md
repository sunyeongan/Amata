# Amata
위치기반 ToDoList
### 프로젝트 배경 
---
"다이소에서 돌돌이 사야겠다~" 생각만 하고 집으로 돌아온 당신을 위한 위치기반 투두리스트
### 프로젝트 설명
---
1. 특정 장소에서 내가 할 일을 적고, 그 장소 주변에 내가 있다면 그 할 일을 푸시 알림 해준다.
2. 내가 가고싶은 여행지에서의 todo도 기록
